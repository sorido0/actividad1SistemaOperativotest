Qué es un Sistema Operativo

Es un software fundamental que actúa como intermediario entre el hardware del computador y los programas del usuario, gestionando recursos y permitiendo la interacción con el sistema.

2. Componentes de un Sistema Operativo

El núcleo (kernel), el elemento central. ... Interpretación de comandos (shell). ... El sistema de archivos (File System) para la gestión de datos, que permite almacenar los archivos en una estructura de árbol.
